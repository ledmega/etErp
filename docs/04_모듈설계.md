# 04. 모듈 설계

## 1. 시스템 관리 모듈 (System Management)

### 1-1. 사용자 관리
| 기능 | 설명 |
|------|------|
| 사용자 목록 조회 | 검색/필터/페이징 |
| 사용자 등록 | 이름, 아이디, 이메일, 부서, 역할 지정 |
| 사용자 수정 | 정보 변경 (비밀번호 별도) |
| 사용자 비활성화 | 논리 삭제 (deleted_yn = 'Y') |
| 비밀번호 초기화 | 임시 비밀번호 발급 |
| 역할 부여/회수 | 다중 역할 지원 |

### 1-2. 권한(역할) 관리
| 기능 | 설명 |
|------|------|
| 역할 목록 조회 | ADMIN, HR, ACCOUNTING, PURCHASE, SALES 등 |
| 역할 등록/수정 | 역할명, 설명 |
| 역할별 메뉴 권한 부여 | 메뉴 접근 허용/차단 |

### 1-3. 메뉴 관리
| 기능 | 설명 |
|------|------|
| 메뉴 트리 조회 | 계층형 메뉴 구조 |
| 메뉴 등록/수정 | URL, 아이콘, 순서, 상위메뉴 지정 |
| 메뉴 활성화/비활성화 | 운영 중 메뉴 노출 제어 |

### 1-4. 공통코드 관리
| 기능 | 설명 |
|------|------|
| 코드 그룹 관리 | 코드 분류 단위 (예: 부서코드, 직급코드) |
| 코드 항목 관리 | 개별 코드값 등록/수정/순서 변경 |

### 1-5. 시스템 로그
| 기능 | 설명 |
|------|------|
| 접속 로그 조회 | 로그인/로그아웃 이력 |
| 작업 감사 로그 | 데이터 생성/수정/삭제 이력 |

---

## 2. 인사관리 모듈 (HR)

### 2-1. 직원 관리
| 기능 | 설명 |
|------|------|
| 직원 목록 조회 | 부서/직급/재직상태별 필터 |
| 직원 등록 | 입사 처리 (기본정보, 학력, 경력) |
| 직원 수정 | 정보 변경 이력 관리 |
| 직원 퇴사 처리 | 퇴사일, 사유 기록 |
| 발령 처리 | 부서이동, 직급변경 이력 |
| 인사카드 출력 | 개인 인사 정보 PDF |

### 2-2. 부서 관리
| 기능 | 설명 |
|------|------|
| 부서 트리 조회 | 조직도 형태 표시 |
| 부서 등록/수정 | 부서명, 상위부서, 부서장 |
| 부서 인원 현황 | 부서별 소속 직원 수 |

### 2-3. 근태 관리
| 기능 | 설명 |
|------|------|
| 출퇴근 기록 | 일별 출퇴근 시간 |
| 초과근무 신청/승인 | 사전 신청 → 팀장 승인 |
| 휴가 신청/승인 | 연차, 반차, 특별휴가 |
| 월별 근태 현황 | 직원별 월간 근태 집계 |
| 잔여 연차 조회 | 개인별 연차 현황 |

### 2-4. 급여 관리
| 기능 | 설명 |
|------|------|
| 급여 항목 설정 | 기본급, 수당, 공제 항목 |
| 월별 급여 계산 | 근태 연동 자동 계산 |
| 급여 명세서 | 개인별 명세서 조회/출력 |
| 급여 대장 | 전체 급여 집계 |

---

## 3. 재무/회계 모듈 (Accounting)

### 3-1. 계정과목 관리
| 기능 | 설명 |
|------|------|
| 계정과목 트리 | 재무상태표/손익계산서 구조 |
| 계정과목 등록/수정 | 계정코드, 계정명, 유형 |

### 3-2. 전표 관리
| 기능 | 설명 |
|------|------|
| 전표 입력 | 분개 입력 (차변/대변 자동 검증) |
| 전표 목록 조회 | 기간/계정/부서별 검색 |
| 전표 승인/반려 | 결재 흐름 |
| 전표 취소 | 역분개 처리 |

### 3-3. 예산 관리
| 기능 | 설명 |
|------|------|
| 연간 예산 편성 | 부서별/계정별 예산 입력 |
| 예산 집행 현황 | 예산 대비 실적 비교 |
| 예산 초과 알림 | 임계치 초과 시 경고 |

### 3-4. 재무 보고서
| 기능 | 설명 |
|------|------|
| 재무상태표 | 자산/부채/자본 |
| 손익계산서 | 수익/비용 |
| 현금흐름표 | 현금 유입/유출 |

## 4. 구매관리 모듈 (Purchase)

## 5. 영업/판매 모듈 (Sales)

| 기능 | 설명 |
|------|------|

| 수주 관리 | 수주 등록 → 납품 처리 |

## 6. 재고관리 모듈 (Inventory)

| 기능 | 설명 |
|------|------|

| 입출고 처리 | 구매입고, 판매출고, 이동 |
