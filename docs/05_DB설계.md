# 05. DB 설계

## 1. 설계 원칙

- 모든 테이블에 **공통 감사 컬럼** 포함 (BaseEntity 상속)
- **논리 삭제** 원칙 (`deleted_yn = 'N'` 기본값, 물리 삭제 금지)
- **Snake_case** 컬럼명 사용
- PK는 **BIGSERIAL** (자동 증가) 또는 **UUID** 사용
- 외래키 제약조건 명시적 정의

---

## 2. 공통 컬럼 (BaseEntity)

모든 테이블에 아래 컬럼이 포함됩니다.

```sql
created_at   TIMESTAMP NOT NULL DEFAULT NOW()  -- 생성일시
created_by   VARCHAR(50)                        -- 생성자 (사용자 ID)
updated_at   TIMESTAMP                          -- 수정일시
updated_by   VARCHAR(50)                        -- 수정자 (사용자 ID)
deleted_yn   CHAR(1) NOT NULL DEFAULT 'N'       -- 삭제여부 (Y/N)
```

## 3. 시스템 관리 테이블

### sys_user (사용자)
```sql
CREATE TABLE sys_user (
    user_id      BIGSERIAL    PRIMARY KEY,
    login_id     VARCHAR(50)  NOT NULL UNIQUE,   -- 로그인 아이디
    password     VARCHAR(255) NOT NULL,           -- BCrypt 암호화
    user_name    VARCHAR(100) NOT NULL,           -- 사용자 이름
    email        VARCHAR(200),
    phone        VARCHAR(20),
    dept_id      BIGINT       REFERENCES hr_dept(dept_id),
    use_yn       CHAR(1)      NOT NULL DEFAULT 'Y',
    last_login_at TIMESTAMP,
    -- 공통 컬럼
    created_at   TIMESTAMP    NOT NULL DEFAULT NOW(),
    created_by   VARCHAR(50),
    updated_at   TIMESTAMP,
    updated_by   VARCHAR(50),
    deleted_yn   CHAR(1)      NOT NULL DEFAULT 'N'
);
```

### sys_role (역할)
```sql
CREATE TABLE sys_role (
    role_id      BIGSERIAL    PRIMARY KEY,
    role_code    VARCHAR(50)  NOT NULL UNIQUE,   -- ROLE_ADMIN, ROLE_HR ...
    role_name    VARCHAR(100) NOT NULL,
    description  VARCHAR(500),
    created_at   TIMESTAMP    NOT NULL DEFAULT NOW(),
    created_by   VARCHAR(50),
    updated_at   TIMESTAMP,
    updated_by   VARCHAR(50),
    deleted_yn   CHAR(1)      NOT NULL DEFAULT 'N'
);
```

### sys_user_role (사용자-역할 매핑)
```sql
CREATE TABLE sys_user_role (
    user_role_id BIGSERIAL PRIMARY KEY,
    user_id      BIGINT NOT NULL REFERENCES sys_user(user_id),
    role_id      BIGINT NOT NULL REFERENCES sys_role(role_id),
    UNIQUE (user_id, role_id)
);
```

### sys_menu (메뉴)
```sql
CREATE TABLE sys_menu (
    menu_id      BIGSERIAL    PRIMARY KEY,
    parent_id    BIGINT       REFERENCES sys_menu(menu_id),  -- 상위 메뉴
    menu_name    VARCHAR(100) NOT NULL,
    menu_url     VARCHAR(200),
    icon_class   VARCHAR(100),
    sort_order   INT          NOT NULL DEFAULT 0,
    use_yn       CHAR(1)      NOT NULL DEFAULT 'Y',
    created_at   TIMESTAMP    NOT NULL DEFAULT NOW(),
    created_by   VARCHAR(50),
    updated_at   TIMESTAMP,
    updated_by   VARCHAR(50),
    deleted_yn   CHAR(1)      NOT NULL DEFAULT 'N'
);
```

### sys_role_menu (역할별 메뉴 권한)
```sql
CREATE TABLE sys_role_menu (
    role_menu_id BIGSERIAL PRIMARY KEY,
    role_id      BIGINT NOT NULL REFERENCES sys_role(role_id),
    menu_id      BIGINT NOT NULL REFERENCES sys_menu(menu_id),
    UNIQUE (role_id, menu_id)
);
```

### sys_common_code (공통코드 그룹)
```sql
CREATE TABLE sys_common_code (
    code_group   VARCHAR(50)  NOT NULL,
    code_value   VARCHAR(50)  NOT NULL,
    code_name    VARCHAR(100) NOT NULL,
    sort_order   INT          NOT NULL DEFAULT 0,
    use_yn       CHAR(1)      NOT NULL DEFAULT 'Y',
    description  VARCHAR(500),
    PRIMARY KEY (code_group, code_value)
);
```

## 4. 인사관리 테이블

### hr_dept (부서)
```sql
CREATE TABLE hr_dept (
    dept_id      BIGSERIAL    PRIMARY KEY,
    parent_id    BIGINT       REFERENCES hr_dept(dept_id),
    dept_code    VARCHAR(20)  NOT NULL UNIQUE,
    dept_name    VARCHAR(100) NOT NULL,
    manager_id   BIGINT       REFERENCES hr_employee(emp_id),
    sort_order   INT          NOT NULL DEFAULT 0,
    use_yn       CHAR(1)      NOT NULL DEFAULT 'Y',
    created_at   TIMESTAMP    NOT NULL DEFAULT NOW(),
    created_by   VARCHAR(50),
    updated_at   TIMESTAMP,
    updated_by   VARCHAR(50),
    deleted_yn   CHAR(1)      NOT NULL DEFAULT 'N'
);
```

### hr_employee (직원)
```sql
CREATE TABLE hr_employee (
    emp_id       BIGSERIAL    PRIMARY KEY,
    emp_no       VARCHAR(20)  NOT NULL UNIQUE,   -- 사번
    emp_name     VARCHAR(100) NOT NULL,
    dept_id      BIGINT       REFERENCES hr_dept(dept_id),
    position_code VARCHAR(20),                   -- 직급 (공통코드)
    job_title    VARCHAR(100),                   -- 직책
    hire_date    DATE         NOT NULL,
    resign_date  DATE,
    emp_status   CHAR(1)      NOT NULL DEFAULT 'A', -- A:재직, R:퇴직
    email        VARCHAR(200),
    phone        VARCHAR(20),
    mobile       VARCHAR(20),
    birth_date   DATE,
    gender       CHAR(1),                        -- M/F
    address      VARCHAR(500),
    bank_code    VARCHAR(10),                    -- 은행코드
    bank_account VARCHAR(50),                    -- 계좌번호
    created_at   TIMESTAMP    NOT NULL DEFAULT NOW(),
    created_by   VARCHAR(50),
    updated_at   TIMESTAMP,
    updated_by   VARCHAR(50),
    deleted_yn   CHAR(1)      NOT NULL DEFAULT 'N'
);
```

### hr_attendance (근태)
```sql
CREATE TABLE hr_attendance (
    att_id       BIGSERIAL    PRIMARY KEY,
    emp_id       BIGINT       NOT NULL REFERENCES hr_employee(emp_id),
    att_date     DATE         NOT NULL,
    check_in     TIMESTAMP,
    check_out    TIMESTAMP,
    att_type     VARCHAR(20)  NOT NULL DEFAULT 'NORMAL', -- NORMAL/LATE/ABSENT/LEAVE
    overtime_hours NUMERIC(4,1) DEFAULT 0,
    note         VARCHAR(500),
    created_at   TIMESTAMP    NOT NULL DEFAULT NOW(),
    created_by   VARCHAR(50),
    updated_at   TIMESTAMP,
    updated_by   VARCHAR(50),
    deleted_yn   CHAR(1)      NOT NULL DEFAULT 'N',
    UNIQUE (emp_id, att_date)
);
```

### hr_leave (휴가)
```sql
CREATE TABLE hr_leave (
    leave_id     BIGSERIAL    PRIMARY KEY,
    emp_id       BIGINT       NOT NULL REFERENCES hr_employee(emp_id),
    leave_type   VARCHAR(20)  NOT NULL,    -- ANNUAL/HALF/SPECIAL
    start_date   DATE         NOT NULL,
    end_date     DATE         NOT NULL,
    leave_days   NUMERIC(4,1) NOT NULL,
    reason       VARCHAR(500),
    status       VARCHAR(20)  NOT NULL DEFAULT 'PENDING', -- PENDING/APPROVED/REJECTED
    approver_id  BIGINT       REFERENCES hr_employee(emp_id),
    approved_at  TIMESTAMP,
    created_at   TIMESTAMP    NOT NULL DEFAULT NOW(),
    created_by   VARCHAR(50),
    updated_at   TIMESTAMP,
    updated_by   VARCHAR(50),
    deleted_yn   CHAR(1)      NOT NULL DEFAULT 'N'
);
```

## 5. 재무/회계 테이블

### acc_account (계정과목)
```sql
CREATE TABLE acc_account (
    account_id   BIGSERIAL    PRIMARY KEY,
    parent_id    BIGINT       REFERENCES acc_account(account_id),
    account_code VARCHAR(20)  NOT NULL UNIQUE,
    account_name VARCHAR(100) NOT NULL,
    account_type VARCHAR(20)  NOT NULL,  -- ASSET/LIABILITY/EQUITY/REVENUE/EXPENSE
    use_yn       CHAR(1)      NOT NULL DEFAULT 'Y',
    created_at   TIMESTAMP    NOT NULL DEFAULT NOW(),
    created_by   VARCHAR(50),
    updated_at   TIMESTAMP,
    updated_by   VARCHAR(50),
    deleted_yn   CHAR(1)      NOT NULL DEFAULT 'N'
);
```

### acc_voucher (전표 헤더)
```sql
CREATE TABLE acc_voucher (
    voucher_id   BIGSERIAL    PRIMARY KEY,
    voucher_no   VARCHAR(30)  NOT NULL UNIQUE,  -- 전표번호 (자동생성)
    voucher_date DATE         NOT NULL,
    voucher_type VARCHAR(20)  NOT NULL,         -- GENERAL/PURCHASE/SALES/CASH
    description  VARCHAR(500),
    status       VARCHAR(20)  NOT NULL DEFAULT 'DRAFT', -- DRAFT/APPROVED/CANCELLED
    dept_id      BIGINT       REFERENCES hr_dept(dept_id),
    total_amount NUMERIC(18,2) NOT NULL DEFAULT 0,
    created_at   TIMESTAMP    NOT NULL DEFAULT NOW(),
    created_by   VARCHAR(50),
    updated_at   TIMESTAMP,
    updated_by   VARCHAR(50),
    deleted_yn   CHAR(1)      NOT NULL DEFAULT 'N'
);
```

### acc_voucher_line (전표 라인)
```sql
CREATE TABLE acc_voucher_line (
    line_id      BIGSERIAL    PRIMARY KEY,
    voucher_id   BIGINT       NOT NULL REFERENCES acc_voucher(voucher_id),
    line_no      INT          NOT NULL,
    account_id   BIGINT       NOT NULL REFERENCES acc_account(account_id),
    debit_amount  NUMERIC(18,2) NOT NULL DEFAULT 0,   -- 차변
    credit_amount NUMERIC(18,2) NOT NULL DEFAULT 0,   -- 대변
    description  VARCHAR(500),
    created_at   TIMESTAMP    NOT NULL DEFAULT NOW(),
    created_by   VARCHAR(50)
);
```

## 6. 구매/재고 테이블

### pur_vendor (거래처)
```sql
CREATE TABLE pur_vendor (
    vendor_id    BIGSERIAL    PRIMARY KEY,
    vendor_code  VARCHAR(20)  NOT NULL UNIQUE,
    vendor_name  VARCHAR(200) NOT NULL,
    biz_no       VARCHAR(20),    -- 사업자번호
    ceo_name     VARCHAR(100),
    phone        VARCHAR(20),
    email        VARCHAR(200),
    address      VARCHAR(500),
    use_yn       CHAR(1)      NOT NULL DEFAULT 'Y',
    created_at   TIMESTAMP    NOT NULL DEFAULT NOW(),
    created_by   VARCHAR(50),
    updated_at   TIMESTAMP,
    updated_by   VARCHAR(50),
    deleted_yn   CHAR(1)      NOT NULL DEFAULT 'N'
);
```

### inv_item (품목)
```sql
CREATE TABLE inv_item (
    item_id      BIGSERIAL    PRIMARY KEY,
    item_code    VARCHAR(20)  NOT NULL UNIQUE,
    item_name    VARCHAR(200) NOT NULL,
    item_type    VARCHAR(20)  NOT NULL,   -- PRODUCT/MATERIAL/SEMI
    unit         VARCHAR(20)  NOT NULL,   -- 단위 (EA, KG, M ...)
    unit_price   NUMERIC(18,2),
    safety_qty   NUMERIC(14,3) DEFAULT 0, -- 안전재고
    use_yn       CHAR(1)      NOT NULL DEFAULT 'Y',
    created_at   TIMESTAMP    NOT NULL DEFAULT NOW(),
    created_by   VARCHAR(50),
    updated_at   TIMESTAMP,
    updated_by   VARCHAR(50),
    deleted_yn   CHAR(1)      NOT NULL DEFAULT 'N'
);
```

### inv_stock (재고 현황)
```sql
CREATE TABLE inv_stock (
    stock_id     BIGSERIAL    PRIMARY KEY,
    item_id      BIGINT       NOT NULL REFERENCES inv_item(item_id),
    warehouse_id BIGINT       NOT NULL,
    current_qty  NUMERIC(14,3) NOT NULL DEFAULT 0,
    updated_at   TIMESTAMP    NOT NULL DEFAULT NOW(),
    UNIQUE (item_id, warehouse_id)
);
```

CREATE INDEX idx_sys_user_login_id  ON sys_user(login_id);
CREATE INDEX idx_hr_employee_emp_no ON hr_employee(emp_no);
