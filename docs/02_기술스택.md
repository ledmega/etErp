### ORM & Query (R2DBC 기반)

| 기술 | 버전 | 용도 |
|------|------|------|
| **R2DBC** | 1.0.x | Reactive Relational DB Connectivity 표준 |
| **r2dbc-mariadb** | 1.2.2 | MariaDB R2DBC 드라이버 |
| **Spring Data R2DBC** | (Boot 내장) | Repository 추상화 (ReactiveCrudRepository) |
| **DatabaseClient** | (Boot 내장) | 복잡 SQL 직접 실행 (QueryDSL 대체) |
| **HikariCP (JDBC)** | (Boot 내장) | Flyway 마이그레이션 전용 JDBC 커넥션 |

> ⚠️ **R2DBC 주의사항**:
> - JPA의 `@OneToMany`, `@ManyToOne` 등 **연관관계 매핑 미지원**
> - 복잡한 JOIN은 `DatabaseClient` 또는 `@Query` SQL로 직접 작성
> - Lazy Loading 없음 → 필요한 데이터는 별도 쿼리로 조회

## 6. 핵심 의존성 요약 (build.gradle.kts)

```kotlin
dependencies {
    // 리액티브 웹 (Tomcat 제외, Netty 사용)
    implementation("org.springframework.boot:spring-boot-starter-webflux")

    // 리액티브 DB (JPA 대신 R2DBC)
    implementation("org.springframework.boot:spring-boot-starter-data-r2dbc")
    runtimeOnly("org.mariadb:r2dbc-mariadb:1.2.2")

    // Flyway 마이그레이션용 JDBC (DDL 실행용)
    implementation("org.flywaydb:flyway-mysql")
    runtimeOnly("org.mariadb.jdbc:mariadb-java-client:3.5.1")

    // 리액티브 Security
    implementation("org.springframework.boot:spring-boot-starter-security")

    // 리액티브 Redis
    implementation("org.springframework.boot:spring-boot-starter-data-redis-reactive")

    // Thymeleaf
    implementation("org.springframework.boot:spring-boot-starter-thymeleaf")
    implementation("org.thymeleaf.extras:thymeleaf-extras-springsecurity6")

    // JWT
    implementation("io.jsonwebtoken:jjwt-api:0.12.6")
    runtimeOnly("io.jsonwebtoken:jjwt-impl:0.12.6")
    runtimeOnly("io.jsonwebtoken:jjwt-jackson:0.12.6")

    // API 문서 (WebFlux 전용)
    implementation("org.springdoc:springdoc-openapi-starter-webflux-ui:2.8.3")

    // Validation
    implementation("org.springframework.boot:spring-boot-starter-validation")

    // Lombok
    compileOnly("org.projectlombok:lombok")
    annotationProcessor("org.projectlombok:lombok")

    // 테스트
    testImplementation("org.springframework.boot:spring-boot-starter-test")
    testImplementation("io.projectreactor:reactor-test")         // StepVerifier
    testImplementation("org.springframework.security:spring-security-test")
}
```

## 7. MVC vs WebFlux 비교 (학습용)

| 항목 | Spring MVC | Spring WebFlux |
|------|------------|----------------|
| **프로그래밍 모델** | 명령형 (Imperative) | 선언형 (Declarative) |
| **서버** | Tomcat (스레드 풀) | Netty (이벤트 루프) |
| **DB 접근** | JPA / JDBC (블로킹) | R2DBC (논블로킹) |
| **반환 타입** | `String`, `List<T>` | `Mono<T>`, `Flux<T>` |
| **스레드** | 요청당 1 스레드 | 적은 스레드 공유 |
| **적합한 상황** | CRUD 위주 단순 앱 | 높은 동시성, I/O 집약적 |
| **학습 난이도** | 낮음 | 높음 (Reactive 사고 필요) |
