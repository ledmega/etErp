## 2. JAVA_HOME 환경변수 설정

PowerShell (관리자 권한)에서 실행:

```powershell
[System.Environment]::SetEnvironmentVariable(
  "JAVA_HOME",
  "E:\util\java\openjdk-24.0.2_windows-x64_bin\jdk-24.0.2",
  "Machine"
)

# PATH에 추가
$path = [System.Environment]::GetEnvironmentVariable("PATH", "Machine")
[System.Environment]::SetEnvironmentVariable(
  "PATH",
  "$path;E:\util\java\openjdk-24.0.2_windows-x64_bin\jdk-24.0.2\bin",
  "Machine"
)
```

재시작 후 확인:
```powershell
java -version
# openjdk version "24.0.2" ...
```

## 4. Docker로 로컬 DB 환경 구성

### docker-compose.yml 작성
프로젝트 루트에 `docker-compose.yml` 파일 생성:

```yaml
version: '3.8'

services:
  postgres:
    image: postgres:16-alpine
    container_name: eternp-postgres
    environment:
      POSTGRES_DB: eternp
      POSTGRES_USER: eternp
      POSTGRES_PASSWORD: eternp1234!
    ports:
      - "5432:5432"
    volumes:
      - postgres_data:/var/lib/postgresql/data
    restart: unless-stopped

  redis:
    image: redis:7-alpine
    container_name: eternp-redis
    ports:
      - "6379:6379"
    command: redis-server --requirepass redis1234!
    restart: unless-stopped

volumes:
  postgres_data:
```

### Docker 실행
```powershell
# 프로젝트 루트에서
docker-compose up -d

# 상태 확인
docker-compose ps

# 로그 확인
docker-compose logs -f postgres
```

## 5. application-dev.yml 설정

`src/main/resources/application-dev.yml`:

```yaml
spring:
  datasource:
    url: jdbc:postgresql://localhost:5432/eternp
    username: eternp
    password: eternp1234!
    driver-class-name: org.postgresql.Driver
    hikari:
      maximum-pool-size: 10
      minimum-idle: 5
      connection-timeout: 30000

  jpa:
    hibernate:
      ddl-auto: update          # 개발 환경: update / 운영: validate
    show-sql: true
    properties:
      hibernate:
        format_sql: true
        dialect: org.hibernate.dialect.PostgreSQLDialect

  data:
    redis:
      host: localhost
      port: 6379
      password: redis1234!
      timeout: 3000ms

  thymeleaf:
    cache: false                # 개발 환경: 템플릿 캐시 끄기

logging:
  level:
    com.et.erp: DEBUG
    org.springframework.security: DEBUG
    org.hibernate.SQL: DEBUG
    org.hibernate.type.descriptor.sql: TRACE

springdoc:
  swagger-ui:
    enabled: true              # 개발 환경에서만 활성화
```
